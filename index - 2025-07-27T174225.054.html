الدرس 1: مقدمة في HTML

HTML هي لغة ترميز تُستخدم لبناء هيكل صفحات الويب. تشمل العناصر مثل العناوين والفقرات والروابط.

الدرس 2: أساسيات CSS

CSS تُستخدم لتنسيق صفحات الويب من حيث الألوان، الخطوط، وتوزيع العناصر.

الدرس 3: ما هي JavaScript؟

JavaScript تُستخدم لجعل صفحات الويب تفاعلية، مثل الاستجابة للأزرار وتحميل البيانات.
